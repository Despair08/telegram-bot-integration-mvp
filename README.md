# Telegram Bot: Интеграция с МИС (MVP)

## Описание проекта
Автоматизация процесса передачи заявок от менеджера в Медицинскую Информационную Систему (МИС).
Ранее процесс был ручным и приводил к потере данных и ошибкам при передаче телефонов.

## Цель
Снизить время обработки заявки и исключить ошибки ввода номера телефона (нормализация 8->7, маска).

## Артефакты

### BPMN Схемы
- **AS-IS (Как было):** [Смотреть схему](BPMN/Request_As-Is.svg) | [BPMN](BPMN/Request_As-Is.bpmn)
  - *Проблема:* Ручная пересылка скриншотов, нет валидации данных.
- **TO-BE (Как будет):** [Смотреть схему](BPMN/Request_To-Be.svg) | [BPMN](BPMN/Request_To-Be.bpmn)
  - *Решение:* Telegram-бот валидирует данные, отправляет API-запрос в МИС. Реализовано редактирование полей перед отправкой.
