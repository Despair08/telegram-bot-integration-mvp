# Telegram Bot: Интеграция с МИС (MVP)

## Описание проекта
Автоматизация процесса передачи заявок от менеджера в Медицинскую Информационную Систему (МИС).
Ранее процесс был ручным и приводил к потере данных и ошибкам при передаче телефонов.

## Цель
Снизить время обработки заявки и исключить ошибки ввода номера телефона (нормализация 8->7, маска).
