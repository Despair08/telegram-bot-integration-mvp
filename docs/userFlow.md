## Пользовательский путь

1. **Старт** → Приветствие + кнопка [Создать заявку]
2. **Шаг 1: Филиал** → Выбор из списка (кнопки)
3. **Шаг 2: Канал привлечения** → Выбор из списка (кнопки)
4. **Шаг 3: Телефон** → Ввод текста, валидация и нормализация
5. **Шаг 4: Комментарий** → Ввод текста (обязательно, от 5 символов)
6. **Шаг 5: Предпросмотр** → Сводка данных + кнопка [Редактировать] с выбором поля
7. **Шаг 6: Отправка** → API-запрос в МИС → Успех/Ошибка → Возврат в меню

-----------
На каждом шаге есть кнопки [Назад] и [Сброс].
