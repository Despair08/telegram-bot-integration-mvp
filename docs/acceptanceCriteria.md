## Критерии Приемки

**Шаг 1: Филиал**

- Текст-подсказка + кнопки филиалов
- Список статичный (обновляется 1-2 раза в год)
- [Назад] → меню, [Сброс] → очистка + меню

**Шаг 2: Канал привлечения**

- Текст-подсказка + кнопки каналов
- Список должен обновляться без изменения кода бота
- [Назад] → филиал, [Сброс] → очистка + меню

**Шаг 3: Телефон**

- Валидация: только цифры, `+`, `-`, `()`, пробел
- Длина: 10-11 цифр после очистки
- Нормализация: убираем маски, `8` → `7`
- Отображение: `+7 (XXX) XXX-XX-XX`
- При ошибке — сообщение "Введите номер цифрами" / "Номер должен содержать 11 цифр"

**Шаг 4: Комментарий**

- Обязательно, от 5 символов, до 255
- При ошибке — "Комментарий слишком короткий" / "обязателен"

**Шаг 5: Предпросмотр**

- Сводка всех данных
- Кнопка [Редактировать] → меню выбора поля → переход на нужный шаг → возврат к предпросмотру
- [Подтвердить] → отправка, [Отмена] → возврат в меню

**Шаг 6: Отправка**

- Показывает "Отправляем заявку..."
- Ответ 200 → "Заявка успешно передана" → меню
- Ответ ≠ 200 → "Ошибка создания заявки, попробуйте позже" → меню
